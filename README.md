# SIROOO AI

A mobile application built with Flutter and a Python (FastAPI) backend, using SQLite for data persistence.

## Features
- Real-time chat interface.
- Backend message storage in SQLite.
- Simple AI echo response.

## Getting Started

### Backend Setup
1. Navigate to `backend/`.
2. Activate venv: `.\venv\Scripts\activate`.
3. Install requirements: `pip install -r requirements.txt`.
4. Run server: `uvicorn main:app --reload`.

### Frontend Setup
1. Navigate to `frontend/sirooo_ai/`.
2. Run app: `flutter run`.
